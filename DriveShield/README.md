# 🚗 DriveShield

📍 Accepted at: IEEE VTC 2025  

---

## 📌 Problem
Vehicular networks are vulnerable to jamming attacks, which can disrupt communication and safety systems.

---

## 🧠 Approach
DriveShield uses Federated Learning with local neural networks trained on each vehicle. Model updates are aggregated using FedAvg.

Blockchain and IPFS ensure secure and tamper-resistant communication.

---

## ⚙️ Tech Used
- Federated Learning (FedAvg)
- Feedforward Neural Network (FNN)
- Blockchain
- IPFS
- VANET / 6G

---

## 📊 Results
- Accuracy: up to 87%
- Precision: 0.83–0.87
- F1-score: up to 0.88

---

## 📄 Abstract
DriveShield is a decentralized Federated Learning framework for real-time jamming attack detection in VANETs under 6G communication. It achieves high detection accuracy and robustness using distributed training.

Blockchain and IPFS provide secure model sharing, while real-time simulation improves adaptability. The system outperforms traditional centralized methods in both accuracy and scalability.
