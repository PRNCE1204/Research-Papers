# 🧠 DiabSecure

📍 Accepted at: CITS 2025  

---

## 📌 Problem
Traditional healthcare systems require sharing sensitive patient data, raising privacy concerns.

---

## 🧠 Approach
DiabSecure uses Federated Learning (FL) with TinyML models deployed on wearable devices. Data remains on-device, and only model updates are shared.

Blockchain and IPFS are integrated to ensure secure and tamper-proof communication.

---

## ⚙️ Tech Used
- Federated Learning (FedAvg, FedSGD, FedOpt)
- TinyML (TFLite)
- Blockchain
- IPFS

---

## 📊 Results
- Accuracy: ~76%
- F1-score: ~0.77  
- High precision across all models

---

## 📄 Abstract
DiabSecure is a decentralized framework for real-time diabetes risk prediction using Federated Learning (FL) and wearable devices. It deploys TinyML models directly on devices, ensuring that sensitive health data never leaves the user.

Model updates are aggregated using FedAvg, FedSGD, and FedOpt, achieving strong accuracy and F1-scores. Blockchain and IPFS ensure secure, tamper-proof updates, making the system suitable for real-time and privacy-preserving healthcare applications.
